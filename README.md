# Snackventure
Snackventure is a hand-drawn platformer adventure game set in the Snack Cinematic Universe (SCU) — an original comic universe created by a group of friends.

🎮 About the Game
Play as one of three snack heroes — Nacho Man, Popcorn Man, or Cheeto Man — and battle your way through crunchy levels filled with enemies, collectibles, and platforming challenges. Each hero comes with their own unique stats and a rival villain:

Hero	Villain
Nacho Man	Evil Kurkure
Popcorn Man	The One Chip
Cheeto Man	Taki Man
✏️ Art Style
All character art and sprites are original hand-drawn illustrations made by the SCU creators, giving the game a unique sketchy, comic-book feel.

🕹️ How to Play
Arrow keys / WASD to move

Space / W / ↑ to jump

Stomp enemies by jumping on them

Collect snacks to build your score

Reach the flag to complete each level

Esc to pause, R to restart

🌐 Built With
Pure HTML, CSS and JavaScript — no libraries, no engines, just code.

Feel free to edit the story/lore bits to match the actual SCU backstory once you have it fleshed out! 🍟
